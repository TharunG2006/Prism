# Changelog

All notable changes to the Prism E2EE platform will be documented in this file.

## [1.0.0] - 2026-04-24
### Added
- **True End-to-End Encryption**: Initial release with RSA-2048 and AES-256-GCM.
- **Security Vault**: Secondary authentication layer for sensitive chat data.
- **Group Multicast**: Secure encrypted group messaging system.
- **File Transmission**: E2EE file sharing via Cloudinary.
- **Institutional Branding**: Integration with institutional contact points.
- **Product Roadmap**: Publicly accessible roadmap page at `/roadmap`.
- **Proprietary Licensing**: Transitioned to "All Rights Reserved" license.

### Changed
- **Contact System**: Optimized for direct email and GitHub reporting.
- **Landing UI**: Premium glassmorphic redesign of the homepage.
- **Security Footers**: Consistent legal and informational links across all auth pages.

### Fixed
- **Socket Latency**: Improved real-time status reporting.
- **Key Derivation**: Hardened PBKDF2 iterations for master password hashing.
